# 工作
通过一个额外保留机制来增强SPS资源重选过程
# 协议
1.  Explicit：每辆车明确告知其他车辆重新选择的资源位置
2.  Early：提前预留位置
3.  Repeated：由于提前预留，导致重复发送信息，提高了接收率
# 结论
提高了数据包接收间隔( Packet Inter-Reception，PIR )和包接收率( Packet Reception Ratio，PRR )
